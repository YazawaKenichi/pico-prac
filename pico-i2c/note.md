# AE-AQM0802

## システムインターフェース

読み取りまたは書き込み動作中、以下の２つの８ビットレジスタが使用される。

1. データレジスタ ( DR )
    マスタからのデータを格納する場所？
2. 命令レジスタ ( IR )
    マスタからの命令コードを格納する場所？

## RS ピン
レジスタを選択することができる。

0. 命令レジスタ（書き込み用）
    ビジーフラグとアドレスカウンタを読み取ることも可
1. データレジスタ（書き込み・読み出し用）

R/W ビットと組み合わせることで、

- 命令レジスタ書き込み
- ビジーフラグとアドレスカウンタの読み取り
- データレジスタの読み書き

を切り替えすることができるようになる。

## R/W ピン
読み取りまたは書き込みを選択する。

0. Write
1. Read

## スレーヴアドレス
```0b0111110 ( 0x7C )```

変更不可
